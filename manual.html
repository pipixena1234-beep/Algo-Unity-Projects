Sticky Notes

  • Unity Hub — Always match project’s Unity version with the installed version in Unity Hub to avoid compatibility issues!
  • Assets — Organize, import, and access all your game files—like scripts, textures, and prefabs—efficiently.
  • Inspector Component — View and edit properties of selected GameObjects and components—it's where you control behaviors, visuals, and settings.
  • Mono Behaviour — Define custom behavior for GameObjects—attach them to objects to control actions like movement, input, and interaction.
  • Prefabs — Save and reuse GameObject templates—ideal for spawning enemies, UI elements, or repeated objects consistently across your scenes.
