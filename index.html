What have I learned Unity with Algorithmics?

I am able to create and modify realistic 3D terrains in Unity using Terrain tools—sculpting landscapes, painting textures, adding vegetation and details, and setting up lighting and environmental effects to build immersive game worlds.
I find joy in learning animation, as it allows me to create and control character and object animations using Unity’s Animator system, Animation window, and keyframe techniques. I’ve learned to set up Animator properties to smoothly transition between animation states, such as walking and running.
I applied physics in Unity by using Rigidbody for motion and forces, and Colliders for detecting collisions in my first 3D platformer game—where enemies take damage when hit by a fireball.
In my 3D platformer and 2-player car racing games, I learned to switch between different camera perspectives (first-person and third-person) and used Cinemachine to create smooth, dynamic camera movements, including split-screen setups for more engaging gameplay.
